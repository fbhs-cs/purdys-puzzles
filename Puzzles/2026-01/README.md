# January 2026 Puzzle

```
6, 7, 13, 17, 25, 32, 37, 47, 58, 71, ...
```

The **100th** number in the sequence is `1259`

The **1000th** number in the sequence is `16703`

What is the sum of the digits in the **2026th** number in the sequence?
